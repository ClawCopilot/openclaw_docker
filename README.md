🚀 **5分钟部署** - 一键脚本，无需配置
🌐 **国内镜像** - apt/npm/pip/Rust/Go 全链路加速

## 快速开始

Linux / macOS：
```bash
git clone https://github.com/ClawCopilot/openclaw_docker.git
cd openclaw_docker
chmod +x *.sh
./start.sh
```

Windows (PowerShell)：
```bash
git clone https://github.com/ClawCopilot/openclaw_docker.git
cd openclaw_docker
.\start.ps1
```
